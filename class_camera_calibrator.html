Подробное описание

Класс для нахождения коэффициентов для конкретной линзы (с помощью которой было снято видео)

Объект класса осуществляет чтение кадров/фотографий с шахматной доской в различных местах на кадре и вычисляет коэффициенты матрицы [] линзы TODO: добавить вид этой матрицы (3 х 3)
